LivelyUI Documentation & Demo

LivelyUI adalah framework CSS ringan untuk membuat animasi UI yang hidup dan dinamis.

LivelyUI Installation

Installation Methods

1. CDN (via GitHub)

Tambahkan berikut di <head> HTML Anda:

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/VincentSaerang21/livelyui/main/livelyui.css">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/gh/VincentSaerang21/livelyui/main/livelyui.js"></script>

2. Local Installation

Langkah-langkah:

  1. Download file:
    • livelyui.css
    • livelyui.js
  2. Letakkan file dalam project Anda
  3. Tambahkan ke HTML:
    <link rel="stylesheet" href="path/to/livelyui.css">
    <script src="path/to/livelyui.js"></script>

Contoh Penggunaan Dasar

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="livelyui.css">
  <script src="livelyui.js"></script>
</head>
<body>
  <button class="lui-scale-up">Hover Me!</button>
  <div class="lui-scroll lui-fade-in">Scroll Animasi</div>
</body>
</html>

1. Default Animations

Semua elemen dasar memiliki animasi fade in secara default.

Ini adalah contoh elemen dengan animasi fade in default.

<div>Ini adalah contoh elemen dengan animasi fade in default.</div>

2. Hover Effects

Tombol dan elemen interaktif akan membesar saat di-hover.

<button class="lui-scale-up">Hover Saya!</button>

3. Class-Based Animations

Gunakan class seperti lui-fade-in, lui-slide-left, atau lui-scale-up untuk mengontrol animasi.

Elemen ini muncul dengan slide dari kiri.

<div class="lui-slide-left">Elemen ini muncul dengan slide dari kiri.</div>

4. On-Scroll Animations

Elemen akan muncul saat masuk ke viewport.

Elemen ini muncul saat di-scroll.

<div class="lui-scroll">Elemen ini muncul saat di-scroll.</div>

5. Disable Animations

Gunakan class lui-none untuk menonaktifkan animasi.

Elemen ini tidak memiliki animasi.

<div class="lui-none">Elemen ini tidak memiliki animasi.</div>

6. Utility Classes

Gunakan utility classes untuk mengatur easing, duration, dan delay.

Elemen ini muncul dengan durasi 1 detik dan delay 0.5 detik.

<div class="lui-fade-in lui-duration-1000 lui-delay-500">Elemen ini muncul dengan durasi 1 detik dan delay 0.5 detik.</div>

7. Utility Classes

Duration Options

Class Value Keterangan
.lui-duration-100 0.1s Durasi sangat cepat
.lui-duration-200 0.2s Durasi cepat
.lui-duration-300 0.3s Durasi sedang-cepat
.lui-duration-500 0.5s Durasi standar (default)
.lui-duration-700 0.7s Durasi lambat
.lui-duration-1000 1s Durasi sangat lambat

Delay Options

Class Value Keterangan
.lui-delay-100 0.1s Delay sangat singkat
.lui-delay-200 0.2s Delay singkat
.lui-delay-300 0.3s Delay sedang
.lui-delay-500 0.5s Delay standar
.lui-delay-700 0.7s Delay panjang
.lui-delay-1000 1s Delay sangat panjang

Easing Options

Class Value Kurva
.lui-ease-in ease-in Mulai pelan, akhir cepat
.lui-ease-out ease-out Mulai cepat, akhir pelan
.lui-ease-in-out ease-in-out Pelan di awal dan akhir (default)

Contoh Kombinasi

Animasi ini memiliki:
- Durasi 0.7s
- Delay 0.5s
- Easing ease-out

<div class="lui-fade-in lui-duration-700 lui-delay-500 lui-ease-out"> <p>Animasi dengan kombinasi utility classes</p> </div>
